## Run Jupyter using docker

Docker 기반 Jupyter 실행
유형1  
```bash
docker run -it --rm -p 10000:8888 -v "${PWD}":/home/jovyan/work jupyter/datascience-notebook
```

유형2  
```bash
docker run -p 10000:8888 jupyter/scipy-notebook
```
